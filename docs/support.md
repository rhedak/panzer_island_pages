# Support and bug reports

For support questions and bug reports, send an email to **[panzerisland@proton.me](mailto:panzerisland@proton.me)**.

There is no public issue tracker. Email is the primary contact for all platforms.

---

## Before reporting a bug

Check the [FAQ](faq.md) first. Common issues such as Maria's water-only movement, skill picks being permanent, and the Steam demo not unlocking the full game are answered there.

---

## What to include in a bug report

A useful report makes it faster to reproduce and fix the issue. Include:

- **Platform**: Steam, Google Play, itch.io, and operating system or Android version
- **What happened**: a brief description of the bug
- **What you expected**: what should have happened instead
- **Steps to reproduce**: what you were doing when it occurred, if you can reproduce it
- **Stage and chapter**: if the bug is tied to a specific stage
- **Screenshot or video**: if the bug is visual, a screenshot helps significantly

You do not need all of these for every report. A clear description of what went wrong is always enough to start.

---

## Feature requests and feedback

General feedback and suggestions are also welcome at the same address. There is no guarantee of a response to every message, but all feedback is read.
